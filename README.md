# jquery-insert-text

Insert text to input field.

## Dependencies

* jquery

## Installation

Install from npm:

    $ npm install @kanety/jquery-insert-text --save

## Usage

```html
<textarea>sample text</textarea>
```

```javascript
$('textarea').insertText('INSERTED');
```
